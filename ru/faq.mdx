---

<AccordionGroup>
  <Accordion title="Как включить функцию, что бы сразу показывался правильный ответ?">
    Эта функция называется "Режим обучения". Вы можете включить ее при создании ссылки на тест. В этом режиме после каждого ответа пользователю будет показан правильный ответ.
  </Accordion>

  <Accordion title='Как добавить поле "имя" перед прохождением теста?'>
    Эта функция доступна при создании ссылки на тест. Необходимо выбрать опцию "Запрашивать имя или email перед тестом". В системе нет никакой проверки контента, поэтому в поле можно вводить любое значение (порядковый номер, псевдоним и т.д.).
  </Accordion>

  <Accordion title='Можно ли изменять при каком количестве правильных ответов тест будет считаться пройденным?'>
    Да, это можно задать при создании ссылки на тест. В поле "Порог прохождения" необходимо указать процент правильных ответов, при котором тест будет считаться пройденным. Например, если вы укажете 50%, то тест будет считаться пройденным, если пользователь ответит правильно на 50% или более вопросов.
  </Accordion>

  <Accordion title='Можно ли менять порядок вопросов в тесте?'>
    Да, вы можете менять порядок вопросов в тесте. Для этого существует два способа:
    1. На странице редактирования теста, вы можете перетаскивать вопросы мышкой, чтобы изменить их порядок.
    2. Можно задать случайный порядок вопросов при создании ссылки на тест. Для этого необходимо отметить опцию "Случайный порядок вопросов". В этом случае, при каждом прохождении теста, вопросы будут отображаться в случайном порядке. Также, дополнительно можно задать опцию "Случайный порядок ответов", чтобы ответы на вопросы также отображались в случайном порядке.
  </Accordion>

  <Accordion title='Можно ли сделать так, чтобы вопросы шли в случайном порядке?'>
    Можно задать случайный порядок вопросов при создании ссылки на тест. Для этого необходимо отметить опцию "Случайный порядок вопросов". В этом случае, при каждом прохождении теста, вопросы будут отображаться в случайном порядке. Также, дополнительно можно задать опцию "Случайный порядок ответов", чтобы ответы на вопросы также отображались в случайном порядке.
  </Accordion>

  <Accordion title='Можно ли защитить тест от прохождения если ссылка попала в чужие руки?'>
    Да, это возможно. Для этого, вы можете удалить ссылку на тест на странице с деталямит теста. После этого, ссылка станет недействительной, и никто не сможет пройти тест по этой ссылке.
  </Accordion>

  <Accordion title='Можно ли сделать так, чтобы результаты прохождения теста были доступны только автору теста?'>
    Да, вы можете сделать так, чтобы результаты прохождения теста были доступны только вам. Для этого, при создании ссылки на тест, выберите опцию для опции "Результаты доступны" значение "Автору". В этом случае, результаты прохождения теста будут доступны только вам, и никто другой не сможет их увидеть.
  </Accordion>

  <Accordion title='Можно ли ограничить время на прохождение теста?'>
    Да, вы можете ограничить время на прохождение теста. Для этого при создании ссылки на тест укажите ненулевое значение в поле "Ограничение времени". При нулевом значении ограничение времени не будет применяться. Если пользователь не успевает пройти тест за отведенное время, то ответы будут автоматически отправлены, и тест будет считаться завершенным.
  </Accordion>

  <Accordion title='Как сделать так, чтобы тест был доступен только определенным пользователям?'>
    Вы управляете кому может быть доступен тест, создавая ссылки на тесты. Вы можете создать ссылку, которая будет доступна только тем пользователям, которым вы ее отправите. 
    Существуют дополнительные настройки, которые могут помочь еще более ограничить доступ к тесту:
    1. Когда вы создаете ссылку на тест, где результаты доступны вам, вы можете указать в поле "Количество использований" минимальное необходимое количество использований ссылки. Например, если вы укажете 1, то ссылка будет доступна для прохождения только один раз.
    2. Вы можете задать пароль для ссылки на тест. Для этого, при создании ссылки на тест, отметьте опцию "Требовать пароль" и укажите значение в поле "Пароль". Пользователи должны будут ввести этот пароль, чтобы начать тест.
    Вам необхоимо будет самостоятельно отправить пароль пользователям, которым вы хотите предоставить доступ к тесту.
  </Accordion>

  <Accordion title='Что если пользователь закрыл тест до его завершения и не нажал кнопку "Отправить ответы"?'>
    Ничего страшного, в таком случае, указанные пользователем ответы не будут сохранены, а тест будет завершен автоматически по истечении 3-х часов.
  </Accordion>

  <Accordion title='Можно ли импортировать тест из существующего pdf документа?'>
    Да, вы можете импортировать тест из pdf документа. Для этого, на странице Мои тесты (/my-tests), необходимо кликнуть: "Создать тест" -> "Импортировать вопросы". В появившемся окне выберите pdf файл, в котором содержатся вопросы. После этого система автоматически создаст тест с вопросами из документа. Обратите внимание, что форматирование вопросов может быть не идеальным, и вам может потребоваться внести некоторые правки.

    Помимо этого, вы можете делать скриншоты вопросов и загружать их через чат с ИИ-ассистентом. Для этого, на странице редактирования теста, кликните по иконке "Создать с ИИ" в низу экрана. В открывшемся окне, вы можете вставить в поле для ввода текста скриншот вопроса, и система автоматически попытается распознать текст на изображении и создать вопрос.
  </Accordion>

</AccordionGroup>
